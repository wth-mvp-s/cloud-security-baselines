## Scope
This runbook defines the procedures to follow in case of loss or theft of a mobile device used for Multi-Factor Authentication (MFA).  
It applies to all employees, contractors, and external users with access to company tenants (incl. SAS tenant).  

## Structure
- **01_user_procedure.md** – Immediate user actions  
- **02_it_procedure.md** – IT/Security operational measures  
- **03_legal_dpo_procedure.md** – Legal, GDPR, CNIL compliance steps  
- **Annex A** – User notification template  
- **Annex B** – CNIL notification template  
- **Annex C** – Customer notification letter  
- **Decision Matrix** – Lost vs. Stolen  
- **Traceability** – Ticketing system, status lifecycle
